# birthday-message

This is a simple CLI program to display birthday message.

## How to build and run

Simply clone this repo using the following:

```bash
git clone https://github.com/OrangeJuice0105/birthday-message.git
```

From here, there are two ways to build this:

### Build from source

After cloning, enter the directory and run the following commands:

```bash
cd birthday-message
make all
make run
```

To exit the program, simply type `q` or press `Ctrl+C` to terminate.

### Installation

If you want persistent program for your environment, simply run the following script:

```bash
sudo sh ./script/install.sh
```
